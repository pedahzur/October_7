# October_7

Workflow, data, and analysis for research on the role of special operations forces (SOF)
during the October 7, 2023 attack. Following the consolidation of the
[Evolving-Paradigms](https://github.com/pedahzur/Evolving-Paradigms) repository, this is also
the home of related materials on evolving paradigms in security studies and modern warfare.

**➡️ For the full analysis — interactive map, infographics, and inferential statistics — visit
the project website: [pedahzur.github.io/October_7](https://pedahzur.github.io/October_7/)**
(available in English and Hebrew).

## Contents

- **Locations.csv / Units.csv** — Primary datasets of locations and units.
- **docs/** — Source files for the project website (location-level analysis, cluster-robust
  inference, interactive map, and animations), published via GitHub Pages at the link above.
- **ERC paper/** — Manuscript materials, bibliography, and a Hebrew translation of manuscript sections.
- **Ranking Israeli units by distinct deployment locations during the October 7 attack.pdf** — A summary output.
- **Evolving-Paradigms/** — Materials from the former Evolving-Paradigms repository, merged
  here in July 2026 with full commit history: a dataset of soldiers who fell on October 7,
  2023, compiled from the official IDF casualties page; October 7–8 military fatalities and
  map data; and research-design notes toward a relational database for political violence
  research. See the folder's [README](Evolving-Paradigms/README.md) for the data log and
  design document.

## Method

An analysis pipeline generating descriptive infographics and inferential statistics from the
locations and units data. Results are presented on the
[project website](https://pedahzur.github.io/October_7/).

## Status & reuse

Active research project. Data and code are shared for transparency and reproducibility;
please cite this repository if you use it and check the license before reuse.


## AI-assisted workflow

Model: Claude, used to assist with the analysis pipeline (descriptive infographics and inferential statistics) and with drafting/organizing project materials. Prompt strategy: iterative pipeline development against the Locations.csv/Units.csv datasets. Human validation: underlying location and unit data, and all published statistics and figures, were reviewed by the author before publication on the project website. Not yet validated: no formal independent replication of the inferential results has been conducted to date.
